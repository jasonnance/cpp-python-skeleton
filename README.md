# Python-C++ Project Skeleton

Provides a minimal skeleton for a project that builds a C++ library linked to Boost which is accessible from Python code.  Includes scaffolding for tests using Boost Unit Test.

## Native install

Requirements:
 - cmake v3.12.4+
 - Python 3.7+
 - clang++ 7.0+
 - Boost 1.68.0+
 
To compile:

    cd build
    cmake ..
    make
